## Libraries and Dev Tools

### Libraries

- [bind-keys](https://github.com/pongo/bind-keys) — a simple keyboard shortcut binding library. `TypeScript`
- [secure-compare-native](https://github.com/pongo/secure-compare-native) — constant-time safe comparison using Node.js native crypto.timingSafeEqual. `Node.js`

### Vue

- [useVariants](https://github.com/pongo/useVariants) — Vue composable for cycling through UI variants and toggling feature flags via global keyboard shortcuts.

### Dev tools

- [eslint-plugin-import-no-private](https://github.com/pongo/eslint-plugin-import-no-private) — eslint plugin disallow jsdoc private imports.
